# Cypress Test for Bayzat

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/dardanmustafa/Bayzat.git
   cd "StarLabs - Bayzat"
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

## Running the Tests
1. To run the tests and record video, use:
   ```bash
   npx cypress run
   ```
